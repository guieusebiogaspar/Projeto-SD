O localaddress especificado nos argumentos é o ipv4 da máquina que está a correr o servidor

Correr RMIServer: como argumento é da forma "java RMIServer localAddress"

Correr MesaVoto: como argumentos é da forma "java MesaVoto multicastAddress departamento localaddress"

Correr AdminConsole: como argumentos é da forma "java AdminConsole localaddress"

Correr VotingTerminal: como argumentos é da forma "java AdminConsole multicastAddress"


Se correr num IDE acionar a opção "Allow multiple instances".
